# A Pytorch Implementation of Hierarchical Similarity Alignment neural Network
(HSANet)

Hierarchical Similarity Alignment for Domain Adaptive Ship Detection in SAR
Images

## Prerequisites

  * Python 2.7 or 3.6
  * Pytorch 0.4.0 
  * CUDA 8.0 or higher
